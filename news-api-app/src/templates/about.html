About the Design

Ella X PyTorch Fintech News is a modern web app built to deliver daily fintech news with AI-powered summaries and a professional, Citi-inspired design.

Tech Stack & Libraries

  • Python — Core programming language
  • Flask — Web framework
  • PyTorch — Deep learning framework (via Hugging Face Transformers)
  • Hugging Face Transformers — For AI-powered text summarization
  • NewsAPI — For fetching the latest fintech news (newsapi.org)
  • HTML/CSS — For frontend design and layout
  • Citi Colors — For a professional fintech vibe
  • GitHub Copilot — AI coding assistant for development help

Designed and developed by Ella.
